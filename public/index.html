Description du projet

Dans le but de combattre les fakes news, l'IA School nous a confié de mener ce projet académique afin de détecter les fausses vidéos et images qui pullulent sur internet et les réseaux sociaux. Notre solution sera de produire un modèle de deep learning capable de les détecter. Notre produit sera une application web mobile et une extension sur les navigateurs.
